# 小二上國語｜公開學習層完成紀錄

日期：2026-07-11
範圍：翰林 114 學年度國小二年級上學期國語 12 課

## 這次完成了什麼

每一課現在都有可追溯、可公開使用的兩層資料：

1. 教育雲的課名、逐課生字、認讀字與語詞（共 365 筆）。
2. 翰林「聽 e 聽」對應課次的課文朗讀／領讀播放清單外連。

內頁以三個可重現的欄位呈現：完整詞彙類別、點按中文發音練習、以及出版社朗讀外連。原始音檔不下載、不嵌入、不重製。

## 修正的品質問題

舊的 L02–L12 將課名、詞彙或常識推成故事情節、文意焦點與任務。這不符合來源政策，因此已全部改成不宣稱課文情節的「公開詞彙與朗讀」學習活動。L01 保留國語 Golden 的直式／注音 renderer，但清楚標示為原創朗讀練習，非教材本文。

## 不等於完成的項目

以下仍然是 Phase 2 blocker，不能因公開學習層完成而宣告國語 Milestone B complete：

- 每課合法 `text-structure brief` 或教師核對摘要。
- 依上述 brief 製作的逐課 NotebookLM source pack、簡報／影音、artifact QA 與 media ID。
- 逐課閱讀理解、人物／事件／段落教學與評量的人工品質確認。

下一個最小 vertical slice 是 L01：取得合法 brief → 教師核對 → NotebookLM source pack → artifact QA → unlisted YouTube media ID → production QA。其餘 11 課依同一合格 schema 推進。
