第 3 單元：分數與加減 小三數學

← 返回總覽
認識假分數與帶分數 同分母分數比較大小 同分母分數加減 假分數與帶分數互換

3-1 認識假分數與帶分數

分數可以分成三種：真分數、假分數、帶分數。

真分數
分子 < 分母，值 < 1
例如：$\dfrac{2}{5}$、$\dfrac{3}{7}$
假分數
分子 ≥ 分母，值 ≥ 1
例如：$\dfrac{5}{3}$、$\dfrac{7}{4}$、$\dfrac{5}{5}$
帶分數
整數 + 真分數，值 > 1
例如：$2\dfrac{3}{4}$、$1\dfrac{1}{3}$
📌 分子 = 分母也算假分數！例如 $\dfrac{5}{5} = 1$，仍然是假分數。

3-2 同分母分數大小比較

同分母的分數，分母一樣，只要比分子就好。分子越大，分數越大。

$$\dfrac{2}{7} \;<\; \dfrac{3}{7} \;<\; \dfrac{5}{7}$$
📌 分母都是 7，看分子 2 < 3 < 5 就好。

3-3 同分母分數加減

同分母分數相加減時，分母不變，只把分子相加或相減。

$$\dfrac{2}{7} + \dfrac{3}{7} = \dfrac{5}{7}$$

計算步驟

1
確認分母相同
兩個分數的分母要一樣才能直接加減
2
分子相加減
分母不動，只算分子
3
化簡或轉換
是假分數時，可化為帶分數

3-4 假分數與帶分數互換

假分數 → 帶分數
分子 ÷ 分母 = 商…餘數
商當整數、餘數當新分子
例：$\dfrac{11}{4} = 2\dfrac{3}{4}$
帶分數 → 假分數
整數 × 分母 + 分子
結果當新分子，分母不變
例：$2\dfrac{1}{3} = \dfrac{7}{3}$

💡 解題步驟（點開展開）

同分母分數相加：分母不變、分子相加
  1. 確認分母相同：兩個分數的分母都是 7 ✓
  2. 分子相加：$2 + 3 = 5$
  3. 分母不變：仍然是 7
答：$\dfrac{2}{7} + \dfrac{3}{7} = \dfrac{5}{7}$
用「分子 ÷ 分母」找出整數部分與餘數
  1. 分子 ÷ 分母：$11 \div 4 = 2 \cdots 3$（商 2、餘 3）
  2. 商 2 當整數部分
  3. 餘數 3 當新的分子，分母 4 不變
答：$\dfrac{11}{4} = 2\dfrac{3}{4}$
用「整數 × 分母 + 分子」算出新的分子
  1. 整數 × 分母：$2 \times 3 = 6$
  2. 加上分子：$6 + 1 = 7$
  3. 新分子 7，分母 3 不變
答：$2\dfrac{1}{3} = \dfrac{7}{3}$

🎮 互動實驗：分數圓

調整分子與分母，觀察圓被分成幾片、填了幾片，並判斷它是真分數還是假分數。

$\dfrac{3}{4}$ 是真分數

💡 小發現：當分子 ≥ 分母時，需要超過一個圓才畫得完，這就是假分數！

📚 關鍵名詞

名詞說明
真分數分子比分母小的分數，值 < 1（例：$\dfrac{2}{5}$、$\dfrac{3}{7}$）
假分數分子 ≥ 分母的分數，值 ≥ 1（例：$\dfrac{5}{3}$、$\dfrac{5}{5}$）
帶分數整數 + 真分數，值 > 1（例：$2\dfrac{3}{4}$）
同分母兩個分數的分母相同，可以直接做加減

易錯點提醒

  • 加法時分母也加了（$\dfrac{2}{7} + \dfrac{3}{7}$ 誤算成 $\dfrac{5}{14}$）
  • 假分數化帶分數時商和餘數搞混（$\dfrac{11}{4}$ 寫成 $3\dfrac{1}{4}$）
  • 忘記「分子 = 分母」也是假分數（如 $\dfrac{5}{5}$）
  • 帶分數化假分數時忘記加分子，只算 整數 × 分母
