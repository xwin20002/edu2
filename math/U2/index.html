第 2 單元：公升與毫升 小三數學

認識毫升（mL）與公升（L） 知道 1L = 1000mL 能做公升與毫升的換算 能做容量加減

2-1 認識毫升（mL）

毫升（mL）是比較小的容量單位，用量杯可以量出毫升數。

生活中的毫升

眼藥水
一瓶大約 10 mL
養樂多
一瓶大約 100 mL
鋁箔包飲料
一包大約 300 mL
📌 讀量杯：眼睛要和液面平視，才能讀到正確的刻度。

2-2 認識公升（L）

公升（L）是比較大的容量單位。

生活中的公升

鮮奶
一瓶大約 1 L
寶特瓶
一瓶大約 2 L
水桶
一桶大約 10 L
$$1\ \text{L} = 1000\ \text{mL}$$

2-3 容量換算與加減

核心換算關係

$$1\ \text{L} = 1000\ \text{mL}$$

L → mL（公升變毫升）

$$2\ \text{L}\ 300\ \text{mL} = 2300\ \text{mL}$$

mL → L（毫升變公升）

$$3500\ \text{mL} = 3\ \text{L}\ 500\ \text{mL}$$

L → mL
公升 × 1000
$2\ \text{L} = 2000\ \text{mL}$
2

毫升 ÷ 1000
$3000\ \text{mL} = 3\ \text{L}$

先統一單位再算
mL 滿 1000 進 1 L

💡 解題步驟（點開展開）

已知：$2\ \text{L}\ 300\ \text{mL}$，要全部換成 mL
  1. 先把公升部分換成毫升：$2\ \text{L} = 2 \times 1000 = 2000\ \text{mL}$
  2. 再加上原本的毫升：$2000 + 300 = 2300$
答：$2\ \text{L}\ 300\ \text{mL} = 2300\ \text{mL}$
已知：$3500\ \text{mL}$，要拆成「幾公升幾毫升」
  1. 看 $3500$ 裡面有幾個 $1000$：$3500 \div 1000 = 3 \text{ 餘 } 500$
  2. 商 $3$ 就是公升數，餘數 $500$ 就是剩下的毫升
答：$3500\ \text{mL} = 3\ \text{L}\ 500\ \text{mL}$
計算：$1\ \text{L}\ 500\ \text{mL} + 800\ \text{mL}$
  1. 毫升相加：$500 + 800 = 1300\ \text{mL}$
  2. $1300\ \text{mL}$ 超過 $1000$，要進位：$1300 = 1\ \text{L}\ 300\ \text{mL}$
  3. 公升相加：$1 + 1 = 2\ \text{L}$
  4. 合起來：$2\ \text{L}\ 300\ \text{mL}$
答：$1\ \text{L}\ 500\ \text{mL} + 800\ \text{mL} = 2\ \text{L}\ 300\ \text{mL}$（也就是 $2300\ \text{mL}$）

教學策略

  • 實物操作：帶量杯、寶特瓶實物讓學生實際量水
  • 生活連結：蒐集不同飲料包裝，讓學生找出容量標示
  • 估測練習：先讓學生猜某容器有多少 mL，再實際量測
  • 換算口訣：「公升變毫升，乘以一千；毫升變公升，除以一千」

易錯點提醒

  • 公升和毫升的 $1000$ 倍關係記不住（常誤寫為 $100$ 倍）
  • 混合單位加減忘記借位（如 $3\ \text{L}\ 200\ \text{mL} - 1\ \text{L}\ 500\ \text{mL}$）
  • 毫升相加超過 $1000$ 時，忘記進位到公升
  • 單位符號混淆（mL 與 L 大小寫）
